## Fitur-fitur

### Pilihan Metrik:

1. **Panjang**
   - Milimeter (mm)
   - Centimeter (cm)
   - Desimeter (dm)
   - Meter (m)
   - Dekameter (dam)
   - Hektometer (hm)
   - Kilometer (km)

2. **Massa**
   - Miligram (mg)
   - Centigram (cg)
   - Desigram (dg)
   - Gram (g)
   - Dekagram (dag)
   - Hektogram (hg)
   - Kilogram (kg)

3. **Waktu**
   - Milisekon (ms)
   - Centisekon (cs)
   - Desisekon (ds)
   - Sekon (s)
   - Dekasekon (das)
   - Hektosekon (hs)
   - Kilosekon (ks)

4. **Arus Listrik**
   - Miliampere (mA)
   - Centiampere (cA)
   - Desiampere (dA)
   - Ampere (A)
   - Dekaampere (daA)
   - Hektoampere (hA)
   - Kiloampere (kA)

5. **Suhu**
   - Celcius (°C)
   - Fahrenheit (°F)
   - Kelvin (K)
